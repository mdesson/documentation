Experiential Learning Endpoint

Data from Experiential Learning Endpoint. Catalog of courses with an experiential learning component

Click here to view the data dictionary


Usage

GET /exle/{method name}/filter/{parameters}

Methods

Course Catalog

Description List of courses descriptions. Can be filter by: Falculty,Department,Program,Degree. And ordering field can be provided as well as an offset and a result limiter
Paramenters {*} or {faculty}
{*} or {department}
{*} or {program}
{*} or {degree}
{*} or {type}
{*} or {orderby}
{*} or {offset}
{*} or {count}
Call https://opendata.concordia.ca/API/v1/exle/catalog/filter/{faculty}/{department}/{program}/{degree}/{type}/{orderby}/{offset}/{count}
Example Call [Courses from the Building Engineering program order by title]
https://opendata.concordia.ca/API/v1/exle/catalog/filter/*/*/Building Engineering/*/*/Title/0/100
Response exle.json
