Library Endpoint

Live data on the current status of some of the Library services and events.

Usage

GET /library/{method name}/{parameters}

Methods

Opening Hours

Description Library Opening hours for the given date
Paramenters A date string (YYYY-MM-DD). This date shouldn't be too much ahead in future
Call https://opendata.concordia.ca/API/v1/library/hours/{date}
Response library-opening-hours.json

Upcoming Events

Description Returns the events (Up to 4) taking place at the Library in the near future.
Paramenters None
Call https://opendata.concordia.ca/API/v1/library/events/
Response library-upcoming-events.json

Computer Availability

Description Real time availability of desktop computers, laptops and tablet in Webster and Vanier libraries
Paramenters None
Call https://opendata.concordia.ca/API/v1/library/computers/
Response library-computer-availability.json

Occupancy

Description Real time estimate of people inside the library different buildings based on data collected by the thermal and optical sensors placed in the different doors.
Paramenters None
Call https://opendata.concordia.ca/API/v1/library/occupancy/
Response library-occupancy.json

Room List

Description A list of all “bookable” resources in the library including group study rooms, presentation rooms and scanners. This method is normally use to obtain the resource identifier and schedule identifier needed to make the call to get the real-time availability of the resource.
Paramenters None
Call https://opendata.concordia.ca/API/v1/library/rooms/getRoomsList
Response library-room-list.json

Reservations

Description A real-time list of reservations made in current day for the given resource
Paramenters Resource ID and Schedule ID
Call https://opendata.concordia.ca/API/v1/library/rooms/getRoomReservations/{resourceID}/{scheduleID}
Response library-room-reservations.json
