Library Endpoint

Live data on the current status of some of the Library services and events.

Usage

GET /library/{method name}/{parameters}

Methods

Opening Hours

Description Library Opening hours for the given date
Paramenters A date string (YYYY-MM-DD). This date shouldn't be too much ahead in future
Examples https://opendata.concordia.ca/API/v1/library/hours/2018-04-19
Response library-opening-hours.json

Upcoming Events

Description Returns the events (Up to 4) taking place at the Library in the near future.
Paramenters None
Examples https://open-data-dev.concordia.ca/API/v1/library/events/
Response library-upcoming-events.json

Computer Availability

Description Real time availability of desktop computers, laptops and tablet in Webster and Vanier libraries
Paramenters None
Examples https://open-data-dev.concordia.ca/API/v1/library/computers/
Response library-computer-availability.json
