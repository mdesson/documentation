Courses Endpoint

Data from the SIS system. Updated nightly.

Click here to view the data dictionary


Usage

GET /course/{method name}/filter/{parameters}

Methods

Course Description

Description List of courses descriptions. Can be filter by Course ID
Paramenters {*} to obtain the full list or a {Course ID}
Call https://opendata.concordia.ca/API/v1/course/description/filter/{courseID}}
Example Call [Description of course 000106] https://opendata.concordia.ca/API/v1/course/description/filter/000106
Response courses-description.json

Course Catalog

Description List of courses. Can be filter by Subject, Catalog or Carreer
Paramenters {subject} or {*}/{catalog} or {*}/{carreer} or {*}
Call https://opendata.concordia.ca/API/v1/course/catalog/filter/{subject}/{catalog}/{carreer}
Example Call [All Biology Courses] https://opendata.concordia.ca/API/v1/course/catalog/filter/BIOL/*/*
Response courses-catalogue.json

Course Section

Description Section details for the courses
Paramenters {subject} or {*}/{catalog} or {*}
Call https://opendata.concordia.ca/API/v1/course/section/filter/{subject}/{catalog}
Example Call [Section details for all PHYS Courses] https://opendata.concordia.ca/API/v1/course/section/filter/PHYS/*
Response courses-section.json

Course Schedule

Description Schedule details for the courses
Paramenters {course id} or {*} / {subject} or {*} / {catalog} or {*}
Call https://opendata.concordia.ca/API/v1/course/schedule/filter/{courseid}/{subject}/{catalog}
Example Call [Schedule details for all BIOL Courses] https://opendata.concordia.ca/API/v1/course/schedule/filter/*/BIOL/*
Response courses-schedule.json

Course Sessions

Description Session details for the courses
Paramenters {career} or {*} / {termcode} or {*} / {sessioncode} or {*}
Call https://opendata.concordia.ca/API/v1/course/session/filter/{career}/{termcode}/{sessioncode}
Example Call [Session details for session code 13W] https://opendata.concordia.ca/API/v1/course/session/filter/*/*/13W
Response courses-session.json

Course Faculty

Description Faculty and department information
Paramenters {facultycode} or {*} / {departmentcode} or {*}
Call https://opendata.concordia.ca/API/v1/course/faculty/filter/{facultycode}/{departmentcode}
Example Call [Details on all faculties and departements] https://opendata.concordia.ca/API/v1/course/faculty/filter/*/*
Response courses-faculty.json

Description List of experiential learning activities currently offered at Concordia. Can be filtered by: Falculty, Department, Program, Degree. And ordering field can be provided as well as an offset and a result limiter
Paramenters {*} or {id or ids(separated by '|')}
{*} or {faculty}
{*} or {department}
{*} or {program}
{*} or {degree}
{*} or {type}
{*} or {level}
{*} or {orderby}
{*} or {offset}
{*} or {count}
Call https://opendata.concordia.ca/API/v1/course/exle/filter/{faculty}/{department}/{program}/{degree}/{type}/{orderby}/{offset}/{count}
Example Call [Courses from the Building Engineering program order by title]
https://opendata.concordia.ca/API/v1/course/exle/filter/*/*/*/Building Engineering/*/*/*/Title/0/100
[Courses with ID GCS_12 and GCS_13]
https://opendata.concordia.ca/API/v1/course/exle/filter/GCS_12|GCS_13/*/*/*/*/*/*/Title/0/100
