Courses Endpoint

Explanation of the dataset

Usage

GET /course/{method name}/filter/{parameters}

Methods

Course Description

Description List of courses descriptions. Can be filter by Course ID
Paramenters {*} to obtain the full list or a {Course ID}
Call https://opendata.concordia.ca/API/v1/course/description/filter/{courseID}}
Response courses-description.json

Course Catalog

Description List of courses. Can be filter by Subject, Catalog or Carreer
Paramenters {subject} or {*}/{catalog} or {*}/{carreer} or {*}
Call https://opendata.concordia.ca/API/v1/course/catalog/filter/{subject}/{catalog}/{carreer}
Response courses-catalogue.json

Course Section

Description Section details for the courses
Paramenters {subject} or {*}/{catalog} or {*}
Call https://opendata.concordia.ca/API/v1/course/section/filter/{subject}/{catalog}
Response courses-section.json

Course Schedule

Description Schedule details for the courses
Paramenters {course id} or {*} / {subject} or {*} / {catalog} or {*}
Call https://opendata.concordia.ca/API/v1/course/schedule/filter/{courseid}/{subject}/{catalog}
Response courses-schedule.json

Course Sessions

Description Session details for the courses
Paramenters {career} or {*} / {termcode} or {*} / {sessioncode} or {*}
Call https://opendata.concordia.ca/API/v1/course/session/filter/{career}/{termcode}/{sessioncode}
Response courses-session.json

Course Faculty

Description Faculty and department information
Paramenters {facultycode} or {*} / {departmentcode} or {*}
Call https://opendata.concordia.ca/API/v1/course/faculty/filter/{facultycode}/{departmentcode}
Response courses-faculty.json
